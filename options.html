Custom Extraction Rules

Define specific CSS selectors to extract content from particular websites. When custom rules are active for a site, ONLY content from your specified selectors is extracted - everything else on the page is completely ignored. This provides maximum storage efficiency and precision.

‚ö° How It Works:
Without custom rules: Extracts main content, removes ads/navigation (~2-4KB per page)
With custom rules: Extracts ONLY from your selectors, ignores everything else (~0.5-2KB per page)
Example: YouTube rule with #description-inline-expander saves ONLY video description, not comments/recommendations/sidebar
